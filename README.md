## 🖥️ Running the Project (Local)

로컬 환경에서 프로젝트를 실행하려면 아래 명령어를 차례대로 입력하세요.

### 1. 패키지 설치
```bash
npm install
```

### 2. 개발 서버 시작
```bash
npm run dev
```

### 3. 프로덕션 빌드
```bash
npm run build
```

### 💡 서버가 시작되면 브라우저에서 http://localhost:5173 으로 접속하세요.
